# BMI Calculator

A new Flutter project.
**A simple Body Mass Index (BMI) calculator built with Flutter.**

## Getting Started

## Features

- Input fields for height and weight
- Calculate BMI based on the inputs
- Display the BMI result with an appropriate message
- Responsive UI
